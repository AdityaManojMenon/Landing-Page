Financial Article Analyzer

I developed an advanced Language Model (LLM) to scrape and summarize financial articles from websites and documents, enabling financial analysts to extract critical information more efficiently than with ChatGPT-3.5. This model supports direct file uploads, simplifying data extraction, and incorporates a built-in knowledge base for accessing multiple articles simultaneously. Utilizing FAISS for rapid vector similarity searches and OpenAI embeddings for precise text representation, the model ensures quick and accurate information retrieval. The user-friendly interface, built with Streamlit, allows seamless URL and query input. Security and efficiency are maintained by securely storing the OpenAI API key and processing data in manageable steps. The solution optimizes token limits and reduces costs by processing only relevant data chunks, enhancing both scalability and cost-effectiveness.

Algorithmic Trading model using Unsupervised Learning

This project leverages unsupervised learning techniques to develop a trading strategy based on clustering financial assets using technical indicators. The primary focus is on applying the K-Means clustering algorithm to identify patterns and trends within the stock data of S&P 500 constituents. Technical indicators such as RSI, ATR, MACD, and Bollinger Bands are calculated to represent different aspects of market behavior. The stocks are then ranked based on dollar volume, and the top 150 most liquid stocks are selected for further analysis. The K-Means clustering is applied monthly, and the stocks in the highest momentum cluster are selected for portfolio optimization. The project uses the Efficient Frontier method to determine optimal portfolio weights, aiming to maximize the Sharpe ratio. The performance of this strategy is then compared against the S&P 500 returns. The results, visualized in the provided graph, indicate that the strategy outperforms the S&P 500 index over the observed period. This demonstrates the potential of unsupervised learning and advanced data analytics in developing effective trading strategies.
